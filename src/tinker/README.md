This module handles compatibility of certain [Tinker](https://github.com/Alphalaneous/Tinker) mod features in [Geode](https://geode-sdk.org/) with Gdvc, currently only compatibility with the `Reference Image` feature is planned / being developed.

**Reference Image**
* Gdvc will automatically copy images to the level folder for safe export and rollback to a previous commit, in case a previously used image may have been deleted.
